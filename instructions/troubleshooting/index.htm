Pre-Game Issues

Installation:

Check Installation

During install:

If an error occurs regarding "C:\Program Files\Crisis Wolverine32\Link to the web site.url" just hit ok. It is not essential to the install in any way.

Harmony.txt on your Desktop:

If you a running CW-IG from a shortcut on your desktop, each time you run it, a small file called harmony.txt will be created. This a 16k harmless file and you can delete it without affecting the game (don't delete it while CW-IG is running though). Just realize that if you delete it and play again, it will appear again, so it probably would be easiest to leave it there until you have completed CW-IG.

General Game Issues

Display:

Some computers (seems to be some laptops) have a problem displaying the game full screen. If the game appears squeezed and out of proportion on your monitor, press the F4 key to make it size correctly. Once Windowed (not maximized) you can then hit the F5 screen for sharper 1:1 pixel picture (though this may be too small to enjoy). Also on some machines you can expand the squeezed window full size by pressing Fn+F7

Fonts:

If you start CW-IG and you cannot see any text in the game menus or dialog boxes, chances are you didn't restart after installation. Make sure to save any open files and restart your computer.

Still having problems with Fonts after restart: In your C:\windows\fonts folder remove the fonts RM2000 and RMG2000 (if they are there at all) and copy/paste the following two fonts into your C:\windows\fonts folder (This folder may be different for some windows OSs, just put the fonts into you standard font folder)
RM2000, RMG2000

Cut-Scene Fonts: Some of you have had a hard time reading the font text in the game AVIs. This isn't a glitch, we just didn't make it big enough (if you run them out site of the game they look fine, it's just the resolution the game runs in.) Sorry for the inconvenience, and if it really a big problem on your machine, try running no AVI's from the beginning. The still images we use in the non-avi versions of the cut-scenes are much more clear.


Game Seems to have little music, Opening, Meijers, etc silent:
Some of you have not been able to hear much of the music that we have included with the game (why do you think it is 100mbs?!) You can hear the midi files but none of the MP3s. We don't know how to solve this glitch but try reinstalling CW-IG, and make sure that if it asks you to overwrite Harmony.dll you say yes. If you still don't have any music why don't you open the MP3 Folder in the project folder and listen to some of the toons we created for CW-IG.

Crashing, or slow play on laptops:
If your laptop computer is crashing, or has slow play, it might be because you are playing with the game windowed. If this is the case, try playing it full screen with no programs running in the background. Also if in windowed mode you get squeezed graphics, (which happens on some laptops set to high screen resolutions) you should be able to maximize it by pressing Fn+F7.

Right-side of Chem building is very choppy:

This glitch should not have any impact on progression in the game, though it may be frustrating. If your game is choppy you are probably playing the game windowed (not full screen). If you switch to full screen (F4) you should not have the problem any longer.

Crash Problems in the Markley Underground if you backtrack:

After you beat the Markley beast and get its key there seems to be problems if you backtrack in ways we didn't plan for. (You ain't missing anything). If you can go back to a saved game before you beat the Markley Beast, do that again, and then go straight to the door (straight upward on the map from where you fight him) that his key opens (and go through it and to the next floor), you should be fine.

Glitches in Bus sequence:

If instead of going right you go left until you see two people standing in the road and you touch them, they run after you until the end of the map. This is part of the caterpillar system that we forgot to remove from the map. Normally, they are told to teleport next to you when you enter a map, but since you are in the bus for this one, we told them not to. We just forgot to erase them.

Also, once you charge through the first demon on the bus map, if you turn around and head back a few tiles the demon event will be called again and the game freezes. This is because the bus wants to go through spots but there are only two left in that direction.

Road to Big House:

When entering the shops on the road to the big house, if you press up after returning from the shop window dialog, you can get stuck inside the shop. Push down when you exit the shop window.
