Starting the Game

To begin Crisis Wolverine: Insurrection Green, double-click on RPG_RT in the folder where you installed the game. The program will run, and eventually you will come to this screen:

To start a New Game, highlight "Start" using the Arrow Keys, then press Enter. If you played before, and saved your game, use the Arrow Keys to highlight "Load", and press Enter to go to the loading screen. If you want to close the program, use the Arrow Keys to highlight "End", and press Enter. Below is the loading screen. Look at the numbers below the picture to find out what everything is.

1. File Number - Each save file has its own file number. You can have up to 15 save
files.

2. Leader's Name - The name of the leader of your party.

3. Leader's Level - The current level of the leader of your party on that save file.

4. Leader's HP - The current HP of the leader of your party on that save file.

5. Empty Save File - If a save slot is has not been used, it will look like this.

6. Character Pictures - Pictures of the characters in your group on that save file.

To load a saved game, use the Arrow Keys to highlight the save file that you want to resume, then press Enter.
